# 🚀 Cosmic Defender

¡Un shoot'em up espacial para la terminal con inercia, fuego con gradiente real y jefes! 👾

## ✨ Características

- 🌌 Nave con física de **inercia** para un control fluido.
- 🔥 Disparos y explosiones con **gradiente de fuego** (blanco → amarillo → naranja → rojo → humo).
- 👾 Variedad de enemigos: tanques/élites, kamikazes que se lanzan en picado y más.
- 💥 Jefes con núcleo destructible.
- ⭐ Campo de estrellas con parallax (estrellas lejanas, cercanas y azules) y nebulosas.
- ⚡ Power-ups, escudos, combos y curaciones.
- 🎨 Paleta rica en terminales de 256 colores, con fallback a 8 colores.

## 🚀 Cómo jugar / ejecutar

```bash
python3 space_battle.py
```

> Requiere una terminal con `curses`. Recomendado soporte de 256 colores.

## 🎮 Controles

- **Flechas / WASD**: mover la nave.
- **Espacio**: disparar.
- **Q / Esc**: salir.

## 🛠️ Tecnología

- **Python 3** con **curses** (sin dependencias externas).
- Sistema de partículas y gradientes de color propios.
